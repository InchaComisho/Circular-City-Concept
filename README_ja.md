## 概要

循環都市構想は、既存都市をゼロから建て直すのではなく、段階的に自然循環の内側で機能するシステムへ改修することを提案する。

主要な変革軸：

- 水循環：雨水回収・再生水ネットワーク・地下水涵養
- 熱管理：植物・水・土壌による都市冷却
- エネルギー：分散型再生可能発電・管内小型水力・建材一体型太陽光
- 食料：都市農業・堆肥化・有機物の土壌還元
- 有機物循環：都市廃棄物と土壌再生のループを閉じる

本構想は都市・文明OSと密接に関連しており、そのレトロフィット実装レイヤーとして位置づけられる。
