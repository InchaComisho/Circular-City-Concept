# 循環都市構想 レトロフィット段階モデル
## Level 0からLevel 5までの既存都市改修フレームワーク

---

## この文書の目的

この文書は、現在の一方向型都市から循環都市OSまでの移行を、Level 0からLevel 5までの段階として整理する。各段階は、導入主体、難易度、最初の行動、避けるべき行動を明確にするための目安である。

---

## レベル比較表

| Level | 状態 | 主な循環 | 主体 | 難易度 |
| --- | --- | --- | --- | --- |
| Level 0 | 現在の一方向型都市 | 限定的 | 全主体 | 現状 |
| Level 1 | 家庭・小規模施設の循環 | 雨水、有機物、非常用水 | 個人、施設 | 低 |
| Level 2 | 建物単位の循環 | 中水、緑化、排水熱、センサー | 管理組合、事業者 | 中 |
| Level 3 | 街区単位の循環インフラ | 再生水、雨庭、都市農園、蓄電 | 街区、自治体 | 中〜高 |
| Level 4 | 自治体単位の循環インフラ | 下水処理水、公共施設、災害水網 | 自治体 | 高 |
| Level 5 | 循環都市OS | 水・熱・エネルギー・食料・有機物・防災の統合 | 自治体・市民・事業者 | 最高 |

---

## Level 0: 現在の一方向型都市

雨水は排水され、下水は処理後に放流され、有機物は多くが焼却され、建物は主にエネルギーを消費し、再利用は限定的である。

最初に行うことは、現状の水、熱、電力、有機物、災害時機能を可視化することである。やってはいけないことは、現状把握なしに高価な設備を導入することである。

---

## Level 1: 家庭・小規模施設の循環

雨水タンク、非飲用利用、ベランダ・屋上緑化、コンポスト、非常用水から始める。

| 項目 | 内容 |
| --- | --- |
| 実行主体 | 個人住宅、小規模店舗、学校の一部 |
| 最初の行動 | 雨水タンク、散水、清掃、非常用水 |
| 移行要件 | 誤飲防止、清掃、保守担当 |
| やってはいけないこと | 雨水を飲用化する、管理不能な堆肥を作る |

---

## Level 2: 建物単位の循環

マンション、オフィス、商業施設で、雨水貯留、中水・グレーウォーター、屋上緑化、太陽光統合、排水熱回収、基本センサーを導入する。

| 項目 | 内容 |
| --- | --- |
| 実行主体 | 管理組合、ビル管理者、事業者 |
| 最初の行動 | 共用部雨水利用、トイレ洗浄、植栽灌水 |
| 移行要件 | 配管分離、水質検査、管理規約、保守契約 |
| やってはいけないこと | 管理責任不明の中水設備を入れる |

---

## Level 3: 街区単位の循環インフラ

再生水配管、雨庭、透水性舗装、ミスト冷却、都市農園、分散蓄電、有機物回収を街区として組み合わせる。

| 項目 | 内容 |
| --- | --- |
| 実行主体 | 街区管理者、自治会、開発事業者、自治体 |
| 最初の行動 | 雨庭、公共空間の暑熱対策、有機物回収 |
| 移行要件 | 住民合意、費用負担、水質・衛生管理 |
| やってはいけないこと | ミストや再生水を検証なしに人が多い場所へ導入する |

---

## Level 4: 自治体単位の循環インフラ

高度下水再利用、公共用途の処理水利用、下水熱回収、公共施設BIPV、災害時水ネットワーク、水・熱・エネルギーデータ統合を進める。

| 項目 | 内容 |
| --- | --- |
| 実行主体 | 自治体、上下水道部局、公共施設管理者 |
| 最初の行動 | 公共施設実証、避難所循環化、処理水の非飲用利用 |
| 移行要件 | 条例、監査、第三者評価、公開データ |
| やってはいけないこと | 飲用再利用を合意形成なしに進める |

---

## Level 5: 循環都市OS

AI支援の循環管理により、水、熱、エネルギー、食料、有機物、防災を統合する。適応的保守、公開性、循環ガバナンスを備える。

| 項目 | 内容 |
| --- | --- |
| 実行主体 | 自治体、市民、事業者、研究機関 |
| 最初の行動 | データ標準化、手動運用とAI支援の役割分担 |
| 移行要件 | 透明性、監査、説明可能性、手動切替 |
| やってはいけないこと | AIを最終責任者として扱う |

---

## 移行要件

Levelを上げる条件は、設備数ではなく、安全性、維持管理、合意形成、制度、データの成熟である。

| 移行 | 必要条件 |
| --- | --- |
| 0から1 | 小規模な非飲用循環と保守習慣 |
| 1から2 | 建物管理体制、配管分離、水質確認 |
| 2から3 | 街区合意、費用負担、公共空間の安全管理 |
| 3から4 | 自治体制度、監査、公共インフラ運用 |
| 4から5 | データ統合、透明性、AI支援と人間判断の分離 |

---

## まとめ

循環都市はLevel 5から始まるのではない。Level 1の雨水、非飲用利用、緑化、非常用水、保守記録から始まる。段階を飛ばすほど、技術よりも管理不全がリスクになる。
